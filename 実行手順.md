# シフト管理システム - 実行手順

## 1. ターミナルを開く

macOS の場合：アプリケーション → ユーティリティ → ターミナル

## 2. プロジェクトフォルダに移動

```bash
cd /Users/kotaushi/shift-manager
```

## 3. 依存パッケージをインストール

```bash
npm install
```

※ Node.js が未インストールの場合は、先に [Node.js](https://nodejs.org/) をインストールしてください。

## 4. サーバーを起動

```bash
npm start
```

「シフト管理システムが起動しました: http://localhost:3456」と表示されればOKです。

## 5. ブラウザで開く

ブラウザ（Chrome、Safari など）で次のURLを開いてください：

**http://localhost:3456**

## 6. ログインして使う

| 役割 | ID | パスワード |
|------|-----|-----------|
| 管理者 | admin | admin123 |
| スタッフ | staff1 | staff123 |
| スタッフ | staff2 | staff123 |

---

## トラブルシューティング

### 「npm: command not found」と表示される場合
→ Node.js がインストールされていません。[Node.js](https://nodejs.org/) をダウンロードしてインストールしてください。

### ポート3456が使用中の場合
→ 別のターミナルで同じシステムが起動している可能性があります。そのターミナルを閉じるか、`PORT=8080 npm start` で別のポートで起動できます。
