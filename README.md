# シフト管理システム

**リクエスト回答型**のシフト集約システムです。管理者が募集枠を作成し、スタッフが「○・△・×」で可否を回答。回答は自動的に管理者画面に集約され、スタッフ間では互いに見えません。

## 要件定義

詳細は [docs/requirements.md](docs/requirements.md) を参照してください。

## 主な機能

### 管理者
- **シフト募集（タスク作成）**: いつ・どの時間帯に・何名必要かを指定した募集枠を作成
- **対象指定**: 全スタッフ or 特定グループに一斉通知
- **回答集約**: 全スタッフの回答を一覧で確認（○・△・×・未回答）
- **スタッフ管理**: スタッフの追加

### スタッフ
- **回答入力**: 提示された日程に対して「○・△・×」で可否を入力
- **プライバシー保護**: 他スタッフの回答は閲覧不可

## 起動方法

```bash
# 依存関係のインストール
npm install

# サーバー起動
npm start
```

ブラウザで http://localhost:3456 を開いてください。

## 初期アカウント

| 役割 | ID | パスワード |
|------|-----|-----------|
| 管理者 | admin | admin123 |
| スタッフ | staff1 | staff123 |
| スタッフ | staff2 | staff123 |

## システムフロー

1. **募集開始**: 管理者が募集日程・必要人数・対象を設定
2. **回答入力**: スタッフが専用画面から「○・△・×」で可否を入力
3. **自動集約**: 管理者の管理画面に全スタッフの回答が一覧化
4. **シフト確定**: 管理者が集約結果を確認し、シフトを確定・調整

## 技術スタック

- **バックエンド**: Node.js + Express
- **データ**: JSONファイル（data.json）
- **認証**: セッション（express-session）
- **フロントエンド**: HTML / CSS / JavaScript
